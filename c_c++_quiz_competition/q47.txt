                                What will be the output of the following C++ code?

                                #include <iostream>
                
                                int main(int argc, char const *argv[])
                                {
                                    cout<<"Hello World";
                                    return 0;
                                }

                                (A) Hello World

                                (C) Run-time error

                                (D) Segmentation fault
